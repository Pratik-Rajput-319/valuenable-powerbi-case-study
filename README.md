# ValueEnable – Insurance Analytics Dashboard (Case Study)

⚠️ **Important Disclaimer**  
This project is a **case study created for learning and demonstration purposes only**.  
All data used is **synthetic / anonymized** and does **not represent any real company, client, or internal data**.

## Project Overview
This Power BI dashboard was designed to analyze insurance-related customer data and support data-driven decision-making.

## Key Objectives
- Understand customer demographics
- Analyze policy type distribution
- Identify income segment trends
- Enable interactive filtering using slicers

## Tools & Skills
- Power BI
- DAX (measures & KPIs)
- Data Modeling
- Dashboard Design
